Logger for [Misskey-Web](https://github.com/MissKernel/Misskey-Web)

![](./resources/preview.gif)

## Install
``` shell
$ npm i misskey-web-logger -g
```

## Run
``` shell
$ misskey-web-logger
```

## Use
### Output to a file
``` shell
$ misskey-web-logger > file.txt
```

## Help
``` shell
$ misskey-web-logger -h
```
